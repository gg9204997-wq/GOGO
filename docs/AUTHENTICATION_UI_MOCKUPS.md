### 1.5 Visual Specifications

#### Color Values
- **Background**: #0F0F1A (Deep Dark)
- **Surface**: #1E1E32
- **Primary Purple**: #6366F1
- **Primary Gradient**: #6366F1 → #8B5CF6 (135deg)
- **Text Primary**: #FFFFFF
- **Text Secondary**: #B8B8D0
- **Text Hint**: #6B6B80
- **Border Color**: #2A2A4A
- **Error**: #EF4444
- **Success**: #10B981

#### Typography
- **Welcome Back**: Cairo 32sp SemiBold, #FFFFFF
- **Joojo Chat**: Cairo 22sp Medium, #FFFFFF, letter-spacing 2px
- **Input Labels**: Cairo 12sp Medium, #6B6B80
- **Input Placeholders**: Cairo 14sp Regular, #6B6B80
- **Input Text**: Cairo 16sp Regular, #FFFFFF
- **Button Text**: Cairo 16sp Medium, #FFFFFF
- **Link Text**: Cairo 14sp Regular, #6366F1
- **Forgot Password**: Cairo 12sp Medium, #6366F1

#### Spacing
- **Page Padding**: 24px
- **Section Spacing**: 32px
- **Input Spacing**: 16px
- **Button Margin Top**: 24px
- **Button Margin Bottom**: 32px

#### Dimensions
- **Logo**: 80px x 80px
- **Input Height**: 56px
- **Input Border Radius**: 12px
- **Button Height**: 56px
- **Button Border Radius**: 12px
- **Social Button**: 56px x 56px

#### Shadows
- **Button Shadow**: 0 4px 8px rgba(99, 102, 241, 0.3)
- **Button Elevation**: 0 4px 8px rgba(0, 0, 0, 0.2)
- **Input Focus Glow**: 0 0 20px rgba(99, 102, 241, 0.3)
- **Input Error Glow**: 0 0 20px rgba(239, 68, 68, 0.3)

#### Icons
- **Back Icon**: arrow_back_ios, 24px, #B8B8D0
- **Logo Icon**: chat_bubble_rounded, 80px, Primary Gradient
- **Email Icon**: person_outline, 24px, #6B6B80
- **Password Icon**: lock_outline, 24px, #6B6B80
- **Eye Icon**: visibility/visibility_off, 24px, #6B6B80
- **Google Icon**: 32px, #FFFFFF
- **Facebook Icon**: 32px, #1877F2
- **Apple Icon**: 32px, #FFFFFF

## 15. Implementation Notes

### 15.1 Icon Library
Use Material Icons (Google Material Design Icons)
- Ensure all icons are available
- Use consistent icon sizes
- Apply correct colors

### 15.2 Font Loading
- Load Cairo font from Google Fonts
- Ensure Arabic support
- Handle font loading gracefully

### 15.3 Gradient Implementation
- Use LinearGradient (135deg)
- Primary: #6366F1 → #8B5CF6
- Gold: #D4AF37 → #F4D03F
- Apply to buttons and accents

### 15.4 Glow Effects
- Use BoxShadow with color
- Apply blur radius
- Use spread radius for intensity

### 15.5 RTL Support
- Detect locale (ar/en)
- Mirror layout for Arabic
- Flip directional icons
- Adjust text alignment

### 15.6 Accessibility
- Ensure 4.5:1 contrast ratio
- Provide semantic labels
- Support screen readers
- Minimum 44px touch targets
